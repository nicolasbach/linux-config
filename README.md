Personal Configuration and Setup Script

This is a personal repository. The install script is only intended to work on baseline arch linux and nothing else.

# Packages
```
packages/
├── 01_cmd_basic
├── 02_wm_basic
├── 03_wayland
├── 03_x11
├── 05_desktop
├── 05_flatpaks
├── 10_nvim
└── 10_vim
```

- **cmd_basic**: Basic utilities (must have for me)
- **wm_basic**: essential gui packages I need and don't depend on X11 or Wayland
- **wayland**: packages for wayland installation (in this case hyprland)
- **x11**: packages for X11 Installation (in this case i3)
- **desktop**: Desktop Packages I usually need like mail client or browser
- **flatpaks**: necessary flatpaks
- **nvim**: additional packages need for neovim
- **vim**: additional packages need for vim
